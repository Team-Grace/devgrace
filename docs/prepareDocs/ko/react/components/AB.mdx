import { AB } from '@devgrace/react';

# AB

`AB 테스트`할 때 사용할 수 있는 컴포넌트입니다. 

아래와 같이 올바르게 사용하지 않으면 `null`을 반환합니다.

- 올바르지 않은 컴포넌트 타입을 사용 (AB.CaseA. AB.CaseB)
- 중복된 컴포넌트 타입을 사용
- 자식 컴포넌트 길이가 `2`가 아닌 경우

<br />

## Interface
```tsx
const AB: {
  ({ children }: { children: React.ReactNode }): JSX.Element | null;
  CaseA({ children }: { children: React.ReactNode }): JSX.Element;
  CaseB({ children }: { children: React.ReactNode }): JSX.Element;
}
```

## Usage

```tsx
import { AB } from '@devgrace/react';

const { CaseA, CaseB } = AB;

const Example = () => {
  return (
    <AB>
      <CaseA>
        <div>A</div>
      </CaseA>
      <CaseB>
        <div>B</div>
      </CaseB>
    </AB>
  );
}; 
```
