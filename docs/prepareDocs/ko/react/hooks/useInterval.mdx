# useInterval

`window.setInterval`을 편리하게 사용할 수 있는 커스텀 훅 입니다. 

setInterval과 useInterval 훅의 주요 차이점은 그 인수가 "동적"이라는 것입니다.

<br />

## Interface
```tsx
const useInterval: (callback: () => void, delay?: number) => void
```

## Example

```tsx
import { useInterval } from '@devgrace/react';

const Example = () => {
 const [count, setCount] = useState(0);

  useInterval(() => {
    setCount(count + 1);
  }, 1000);

  return <h1>{count}</h1>;
};
```
