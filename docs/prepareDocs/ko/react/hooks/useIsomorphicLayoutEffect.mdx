# useIsomorphicLayoutEffect

`useIsomorphicLayoutEffect`는 서버 사이드 렌더링 환경에서도 동기적으로 실행되도록 보장하는 hook 입니다.

client환경에서는 useLayoutEffect를 반환하고, server환경에서는 useEffect를 반환합니다.

<br />

## Example

```tsx
import { useIsomorphicLayoutEffect } from '@devgrace/react';

const Example = () => {
  useIsomorphicLayoutEffect(() => {
    console.log(
      "I'm `useLayoutEffect` in the client environment, `useEffect` in the server environment.",
    )
  }, []);
};
