# useIsomorphicLayoutEffect

`useIsomorphicLayoutEffect`는 서버 사이드 렌더링 환경에서도 동기적으로 실행되도록 보장하는 hook 입니다.

<br />

## Example

```tsx
import { useIsomorphicLayoutEffect } from '@devgrace/react';

const Example = () => {
  useIsomorphicLayoutEffect(() => {
    console.log(
      "I'm `useLayoutEffect` OR `useEffect`.",
    )
  }, []);
};
