# isNumber

주어진 인자가 `숫자`인지 검사하고, 맞다면 인자의 타입을 `number`로 좁혀주는 함수입니다.

<br />

## Interface
```tsx title="typescript"
const isNumber: (arg: unknown) => arg is number
```

## Usage
```ts
import { isNumber } from '@devgrace/utils';


isNumber(123); // true

isNumber(() => {}); // false
isNumber('123'); // false
isNumber(true); // false
isNumber({}); // false
isNumber([]); // false
```
